In modern data analysis, data is not always stored in files like CSV or Excel. Very often, data is generated continuously ‚Äî stock prices, weather updates, cryptocurrency values, sports scores, traffic conditions, etc.

This real-time or frequently updated data is accessed using something called an API.

An API (Application Programming Interface) allows one software system to communicate with another. In data analysis, APIs help us:

  • Fetch live data

  • Automate data collection

  • Build dashboards and analytical models

6Ô∏è‚É£ Understanding API Endpoint

An endpoint is simply a URL where the API receives requests.

7Ô∏è‚É£ JSON ‚Äì The Language of APIs

Most APIs return data in JSON (JavaScript Object Notation) format.

JSON looks like a Python dictionary.

Example JSON Response:

json
{
  "city": "London",
  "temperature": 18,
  "humidity": 70
}

In Python, it becomes:

python
{
    "city": "London",
    "temperature": 18,
    "humidity": 70
}

So JSON ‚Üí Python Dictionary

üîü Example: Fetching Cryptocurrency Price (Conceptual)

Let us imagine an API that gives Bitcoin price.

python
import requests

url = "https://api.crypto.com/price?coin=bitcoin"
response = requests.get(url)

price = data["price"]
print("Bitcoin Price:", price)

What is happening?

  1. Python sends GET request

  2. API returns JSON

  3. JSON converted to dictionary

  4. We extract the value using key

1Ô∏è‚É£1Ô∏è‚É£ Handling Errors Properly

A professional data analyst must always check for errors.

python
response = requests.get(url)

if response.status_code == 200:
    data = response.json()
else:
    print("Error fetching data")

Better version:

python
try:
    response = requests.get(url)
    response.raise_for_status()
    data = response.json()
except requests.exceptions.RequestException as e:
    print("API Error:", e)

1Ô∏è‚É£2Ô∏è‚É£ API Key Authentication

Why API Keys?

  • Track usage

  • Enforce rate limits

  • Provide security
