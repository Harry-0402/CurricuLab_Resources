1Ô∏è‚É£ Creating Custom Functions

Introduction

In Unit 1, we learned how to write basic programs using variables, loops, and conditions. But real-world programs are large. Writing everything in one place makes code messy and difficult to manage.

Functions solve this problem.

A function is a reusable block of code that performs a specific task.

Think of a function like a machine:

  • You give it input.

  • It processes the input.

  • It gives output.

Key Concepts

  • Function Definition

  • def keyword

  • Parameters

  • Arguments

  • Return Statement

  • Local vs Global Variables

Detailed Explanation

üîπ 1. Defining a Function

In Python, functions are created using the def keyword.

Syntax:

python
def function_name(parameters):
    # block of code
    return value

üîπ 2. Example: Simple Function

python
def greet():
    print("Hello, Welcome to Data Analysis!")

Calling the function:

python
greet()

Explanation:

  • def tells Python we are defining a function.

  • greet is the function name.

  • Parentheses () may contain parameters.

  • Indentation defines the function body.

üîπ 3. Function with Parameters

Parameters allow input into a function.

python
def greet(name):
    print("Hello", name)

Calling it:

python
greet("Rahul")

Output:

nginx
Hello Rahul

Here:

  • name is a parameter.

  • "Rahul" is an argument.

üîπ 4. Returning Values

The return keyword sends back a result.

python
def add(a, b):
    return a + b

python
result = add(10, 5)
print(result)

Output:

15

Important:

  • return ends function execution.

  • A function can return multiple values using comma separation.

üîπ 5. Local vs Global Variables

python
x = 10   # Global variable

def test():
    y = 5  # Local variable
    print(x + y)

test()

  • Global Variable ‚Üí Defined outside function.

  • Local Variable ‚Üí Defined inside function.

  • Local variables cannot be accessed outside.

Real-World Example: Loan EMI Calculator (Concept)

python
def calculate_emi(principal, rate, time):
    emi = (principal * rate * time) / 100
    return emi

This function can be reused multiple times for different customers in banking systems.

Summary ‚Äì Functions

‚úî Functions improve code reusability
‚úî They make programs modular
‚úî They reduce repetition
‚úî def defines function

2Ô∏è‚É£ Indefinite Loops: While Loops

Introduction

In Unit 1, we learned for loops (definite loops).

A while loop is different.

  • A for loop runs a fixed number of times.

  • A while loop runs as long as a condition is true.

This is called an indefinite loop.

Key Concepts

  • While keyword

  • Condition-based looping

  • Infinite loop

  • Break statement

  • Continue statement

Detailed Explanation

Syntax:

python
while condition:
    # code block

The loop continues until the condition becomes False.

Example 1: Basic While Loop

python
i = 1

while i <= 5:
    print(i)
    i += 1

Output:

1
2
3
4
5

Explanation:

  • Condition checked first.

  • Executes code.

  • Increments value.

  • Stops when condition becomes False.

‚ö† Infinite Loop

If the condition never becomes False:

python
while True:
    print("Hello")

This runs forever.

To stop it, we use break.

Break Statement

python
i = 1

while i <= 10:
    if i == 5:
        break
    print(i)
    i += 1

Stops loop when i = 5.

Continue Statement

Skips current iteration.

python
i = 0

while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)

Output:

1
2
4
5

Flowchart of While Loop

  ‚Üì

  ‚Üì

  ‚Üì False
End

Real-World Example: Password Validation

python
password = ""

while password != "admin123":
    password = input("Enter Password: ")

print("Access Granted")

Used in authentication systems.

Summary ‚Äì While Loop

‚úî Runs until condition is False
‚úî Used when number of iterations is unknown
‚úî Can create infinite loops
‚úî break ‚Üí stops loop
‚úî continue ‚Üí skips iteration

4Ô∏è‚É£ Slicing Data Types

Introduction

Slicing allows us to extract a portion (subset) of data from:

  • Strings

  • Lists

  • Tuples

It is one of the most powerful features in Python.

Key Concepts

  • Indexing

  • Positive Index

  • Negative Index

  • Start : Stop : Step

  • Immutable vs Mutable slicing

üîπ Indexing Basics

Every sequence has positions starting from 0.

Example:

python
text = "PYTHON"
CharacterPYTHON
Index012345
Negative-6-5-4-3-2-1

üîπ Basic Slicing Syntax

python
sequence[start : stop : step]

  • start ‚Üí inclusive

  • stop ‚Üí exclusive

  • step ‚Üí jump interval

Example 1: String Slicing

python
text = "PYTHON"

print(text[0:3])

Output:

nginx
PYT

Because stop index (3) is excluded.

Example 2: List Slicing

python
numbers = [10, 20, 30, 40, 50]

print(numbers[1:4])

Output:

csharp
[20, 30, 40]

Example 3: Step Value

python
print(text[::2])

Output:

nginx
PTO

Explanation:

  • Start at beginning

  • Skip every second character

üîπ Reverse Slicing

python
print(text[::-1])

Output:

nginx
NOHTYP

Slicing Visualization

makefile

Real-World Application

‚úî Extract domain from email
‚úî Extract date from timestamp
‚úî Extract columns from dataset

Example:

python
email = "student@gmail.com"
domain = email[email.index("@")+1:]
print(domain)

Summary ‚Äì Slicing

‚úî Works on strings, lists, tuples
‚úî Uses start:stop:step
‚úî Stop index excluded
‚úî Negative indexing supported

5Ô∏è‚É£ Reading txt Files with Python

Introduction

Data rarely comes directly into code.

Most real-world data is stored in:

  • .txt files

  • .csv files

  • Logs

  • Reports

Python provides built-in functions to read these files.

Key Concepts

  • open()

  • read()

  • readline()

  • readlines()

  • File modes (r, w, a)

  • with statement

üîπ Opening a File

python
file = open("data.txt", "r")

  • "r" ‚Üí read mode

  • "w" ‚Üí write mode

  • "a" ‚Üí append mode

üîπ Reading Entire File

python
file = open("data.txt", "r")
content = file.read()
print(content)
file.close()

üîπ Reading Line by Line

python
file = open("data.txt", "r")

for line in file:
    print(line)

file.close()

üîπ Best Practice: Using with

python
with open("data.txt", "r") as file:
    content = file.read()
    print(content)

Why use with?

‚úî Automatically closes file
‚úî Safer
‚úî Prevents memory leaks

Real-World Example

Reading sales report:

python
with open("sales.txt", "r") as file:
    for line in file:
        print(line.strip())

Used in:

  • Log file analysis

  • Customer data

  • Transaction records

6Ô∏è‚É£ Analyze Data from Text and CSV File

Introduction

Text files contain unstructured data.

CSV files contain structured data:

  • Comma-Separated Values

  • Rows and Columns

Example CSV:

pgsql
Name,Age,City
Rahul,21,Nashik
Amit,22,Mumbai

üîπ Reading CSV without Pandas

python
with open("students.csv", "r") as file:
    for line in file:
        print(line.strip().split(","))

üîπ Using csv Module (Recommended)

python
import csv

with open("students.csv", "r") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

üîπ Converting CSV to List

python
data = []

with open("students.csv", "r") as file:
    reader = csv.reader(file)
    for row in reader:
        data.append(row)

print(data)

Simple Data Analysis Example

python
total_age = 0
count = 0

with open("students.csv", "r") as file:
    reader = csv.reader(file)
    next(reader)  # skip header
    
    for row in reader:
        total_age += int(row[1])
        count += 1

print("Average Age:", total_age/count)
