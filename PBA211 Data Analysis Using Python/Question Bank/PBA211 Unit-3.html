df["Salary"].rank(method="min")

Question 68:

Explain merging and joining of DataFrames in Pandas with suitable examples.

‚úÖ Introduction

In business systems, data is often stored in different tables.

Example:

  • Employee details in one table

  • Salary details in another table

To combine them, we use:

üëâ Merging
üëâ Joining

In Pandas, this is done using merge() and join() functions.

üîπ Example DataFrames

DataFrame 1: Employee Details

python
import pandas as pd

df1 = pd.DataFrame({
    "EmpID": [101, 102, 103],
    "Name": ["Amit", "Priya", "Rahul"]
})

DataFrame 2: Salary Details

python
df2 = pd.DataFrame({
    "EmpID": [101, 102, 104],
    "Salary": [50000, 60000, 70000]
})

üîπ 1Ô∏è‚É£ Merging DataFrames

python
merged_df = pd.merge(df1, df2, on="EmpID")
print(merged_df)

Output:

nginx
   EmpID   Name  Salary
0    101   Amit   50000
1    102  Priya   60000

Explanation:

  • Only matching EmpID values are merged.

  • 103 and 104 are excluded.

This is called inner join (default).

üîπ Types of Merge (Join Types)

1Ô∏è‚É£ Inner Join (Default)

Only matching records.

python
pd.merge(df1, df2, on="EmpID", how="inner")

2Ô∏è‚É£ Left Join

All records from left DataFrame.

python
pd.merge(df1, df2, on="EmpID", how="left")

3Ô∏è‚É£ Right Join

All records from right DataFrame.

python
pd.merge(df1, df2, on="EmpID", how="right")

4Ô∏è‚É£ Outer Join

All records from both DataFrames.

python
pd.merge(df1, df2, on="EmpID", how="outer")

üîπ 2Ô∏è‚É£ Using join() Method

join() is simpler when joining on index.

python
df1.set_index("EmpID").join(
    df2.set_index("EmpID")
)

üîπ Business Example

Suppose company has:

  • Customer table

  • Order table

To analyze total purchases:

python
pd.merge(customers, orders, on="CustomerID")

üîπ Why Merging is Important

‚úî Combine related data
‚úî Build complete reports
‚úî Perform advanced analysis
‚úî Simulate database joins

Without merging:

  • Data remains incomplete

  • Analysis becomes difficult

üîπ Summary Table

Join TypeMeaning
InnerOnly matching rows
LeftAll left rows
RightAll right rows
OuterAll rows from both

‚úÖ Conclusion

Merging and joining in Pandas allow combining multiple DataFrames based on common columns or indexes. The merge() function supports different join types like inner, left, right, and outer joins. These operations are essential in business analytics for integrating data from multiple sources and generating comprehensive reports.

‚úî Tracking sales growth
‚úî Monitoring stock prices
‚úî Observing revenue trends
‚úî Measuring performance over months

üîπ Basic Syntax

python
import matplotlib.pyplot as plt

plt.plot(x_values, y_values)
plt.show()

üîπ Example: Monthly Sales Trend

python
import matplotlib.pyplot as plt

months = ["Jan", "Feb", "Mar", "Apr"]
sales = [10000, 15000, 12000, 18000]

plt.xlabel("Month")
plt.ylabel("Sales")

plt.show()

Creates the line graph.

2Ô∏è‚É£ plt.xlabel()

Labels the X-axis.

3Ô∏è‚É£ plt.ylabel()

Labels the Y-axis.

4Ô∏è‚É£ plt.title()

Adds a title to the graph.

5Ô∏è‚É£ plt.show()

Displays the plot.

‚ùå Graph becomes confusing

With labels:

‚úî Compare performance over time
